How it works

Four databases

IP2X maps IP → coordinate. Genom maps coordinate → place. ASNDB tiny maps IP → owning ASN. IPBlocklist maps IP → threat signals. Each is a single binary blob, parsed in-memory.

Browser-native

The Rust crate runs server-side. This page is a faithful port to ES modules and DataView. No network calls per lookup. The same logic, the same answers.

Offline after load

~100 MB of binaries stream in once. Subsequent lookups are microseconds. The DBs refresh daily on the Pages build.
